# 音乐文件放置说明

请将你的 MP3 音乐文件放在此目录，并命名为：

- `track1.mp3` - 静心禅音
- `track2.mp3` - 藏传颂钵
- `track3.mp3` - 山间流水
- `track4.mp3` - 晨钟暮鼓

## 音乐要求

1. **格式**: MP3
2. **大小**: 建议每首 3-10MB
3. **时长**: 建议 5-15 分钟（可以循环播放）
4. **风格**: 冥想音乐、禅音、自然声音、佛教音乐等

## 获取免费音乐

推荐网站（下载后改名放入此目录）：
- https://pixabay.com/music/search/meditation/
- https://www.bensound.com/royalty-free-music/tag/meditation
- https://mixkit.co/free-stock-music/tag/meditation/

## 打包说明

放入此目录的音乐文件会在构建时自动打包到 APK 中，支持离线播放。
